# 아이디어 스케치: 익명 담벼락 서비스 '우리들의 낙서장'

## 1. 개요
'우리들의 낙서장'은 누구나 쉽게 자신만의 공간(담벼락)을 만들고, 익명의 사용자들이 포스트잇 형태로 자유롭게 메시지를 남길 수 있는 웹 서비스입니다. 복잡한 가입 절차 없이 담벼락 이름과 비밀번호만으로 공간을 생성하며, 감성적이고 따뜻한 디자인을 지향합니다.

## 2. 주요 타겟
- 친구들과의 익명 롤링페이퍼가 필요한 사람
- 특정 주제에 대해 사람들의 짧은 후기나 응원을 모으고 싶은 운영자
- 익명의 따뜻한 메시지를 공유하고 싶은 개인

## 3. 핵심 기능
### 3.1 담벼락 생성 및 관리
- **담벼락 생성**: 유니크한 담벼락 이름(URL 경로로 활용)과 관리용 비밀번호를 설정하여 생성.
- **비밀번호 보호**: 담벼락 주인은 설정한 비밀번호를 통해 악성 게시글 삭제 가능 (향후 기능).

### 3.2 메모(포스트잇) 작성
- **낙서하기**: 담벼락 하단의 플로팅 버튼을 통해 작성 모달 오픈.
- **본인 확인**: 담벼락 생성 시 설정된 비밀번호를 입력해야 작성 가능 (익명성 유지하되 무분별한 접근 방지).
- **작성 항목**:
    - 메시지 내용 (최대 200자)
    - 작성자 이름 (닉네임)
    - 배경 색상 선택 (노랑, 분홍, 민트, 보라, 오렌지 등 파스텔톤)
- **자유 배치**: 생성된 메모는 캔버스 스타일의 배경 위에 랜덤한 위치 혹은 순차적으로 배치되어 실제 담벼락 같은 느낌 전달.

### 3.3 사용자 경험 (UX/UI)
- **감성적 디자인**: 종이 질감의 배경, 포스트잇 스타일의 컴포넌트, 부드러운 그림자 효과.
- **반응형 레이아웃**: 데스크탑에서는 넓은 캔버스 느낌, 모바일에서는 가독성 있게 정렬된 리스트/그리드 방식 혹은 스와이프 가능한 캔버스.
- **애니메이션**: 메모가 생성될 때 부드럽게 나타나는 효과(Fade-in, Scale-up).

## 4. 기술 스택 (확정)
- **Frontend**: Next.js (App Router), Tailwind CSS
- **Backend/DB**: Supabase (PostgreSQL), Prisma (Optional, Supabase Client 직접 사용 가능)
- **Deployment**: Vercel

## 5. 기대 효과
- 간단한 인터페이스로 사용자 진입 장벽 최소화.
- 포스트잇 UI를 통해 아날로그적 감성 제공.
- 실시간 DB 연동으로 즉각적인 피드백 확인.
